Tentang Sitem PANTAS

Sistem PANTAS (Sistem Pelaporan Aduan Tumpukan Sampah) adalah platform yang memudahkan masyarakat melaporkan tumpukan sampah liar melalui aplikasi atau web. Tanpa perlu registrasi, pengguna dapat mengirimkan laporan menggunakan GPS untuk menentukan lokasi, disertai foto sampah, serta informasi mengenai jenis dan tingkat keparahan sampah tersebut. Sistem ini bekerja sama dengan DLHK Riau untuk menangani laporan secara efektif dan menjaga kebersihan lingkungan.

Buka Sistem Pantas di Web Browser

Pengguna dapat mengakses sistem PANTAS pada web browser kesayangan anda yang tersedia di smartphone atau melalui platform web PANTAS yang dapat diakses dari perangkat apapun. Tidak diperlukan registrasi, sehingga pengguna dapat langsung menggunakan layanan tanpa hambatan.

Aktifkan Lokasi

Untuk melaporkan tumpukan sampah dengan akurat, pastikan Anda mengaktifkan lokasi perangkat Anda. Sistem PANTAS akan menggunakan data lokasi untuk menentukan lokasi sampah secara tepat.

  • Aktifkan GPS diponsel anda untuk menentukan lokasi dengan tepat.
  • Lokasi akan digunakan untuk mengirimkan laporan yang akurat.
  • Sistem Pantas akan menjadikan lokasi anda menjadi titik laporan.

Isi Form Laporan

Setelah lokasi Anda aktif, Anda dapat mengisi form laporan yang mencakup jenis sampah, tingkat keparahan, dan mengunggah foto. Form ini akan membantu pihak berwenang menanggapi laporan dengan cepat.

  • Isi Nama Pelapor.
  • Isi Nomor Whatsapp untuk mengirimkan status laporan
  • Pilih Tingkat Keparahan yaitu Ringan,Sedang atau Berat.
  • Unggah foto sampah untuk mendukung laporan.
  • Isi form Deskripsi laporan.
  • Tekan Tombol Submit untuk menyelesaikan laporan.

Punya Pertanyaan? Cek FAQ

Berikut adalah beberapa pertanyaan yang sering diajukan terkait sistem PANTAS.

Bagaimana cara mengakses sistem PANTAS?

Pengguna dapat mengakses sistem PANTAS melalui web browser di perangkat apapun. Tidak diperlukan registrasi untuk mulai melaporkan sampah.

Apakah saya perlu mendaftar untuk menggunakan sistem PANTAS?

Untuk melaporkan sampah, Anda tidak perlu melakukan registrasi. Cukup buka sistem di web browser dan mulai melaporkan dengan GPS dan foto sampah.

Bagaimana cara melaporkan sampah dengan sistem PANTAS?

Setelah mengakses sistem, aktifkan lokasi perangkat Anda, isi form laporan dengan memilih jenis sampah, tingkat keparahan, serta mengunggah foto sampah, lalu kirimkan laporan.

Bagaimana sistem menentukan lokasi sampah?

Sistem menggunakan GPS di perangkat Anda untuk menentukan lokasi secara otomatis dan menandainya pada peta sebagai titik laporan sampah.

Apakah laporan saya akan diproses dengan cepat?

Ya, laporan yang masuk akan segera diteruskan ke pihak yang berwenang seperti DLHK Riau untuk penanganan lebih lanjut, memastikan respons cepat terhadap sampah liar.

Apakah saya bisa melaporkan sampah di lokasi yang tidak terlihat?

Anda bisa melaporkan sampah di lokasi mana saja menggunakan GPS di perangkat Anda, selama lokasi tersebut dapat dijangkau oleh ponsel dan sinyal GPS tersedia.

Bagaimana cara saya mengetahui status laporan saya?

Setelah mengirimkan laporan, Anda dapat mengisi nomor Whatsapp untuk mendapatkan informasi status laporan melalui pesan langsung.
